== Usage ==

Add the Financial Calculator shortcode to your page, post or sidebar:

`[ci_financial_calculator]`

Add the following code to your template where you would like the Financial Calculator to appear:

`<?php display_ci_financial_calculator(); ?>`

== Installation ==

1. Upload the Financial Calculator /ci_financial_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Financial Calculator plugin through the "Plugins" menu in WordPress.
